To use this element in your course, you must access the Github repository for this course. The repository is linked on the PrairieLearn website and contains the element itself, as well as any sample questions and documentation. Follow these steps to use the element:

  1. Clone or download the Github repository for this course.
  2. Copy the content of the elements folder into your own course repository. If your course does not have an elements folder, you need to create one first.
  3. Commit and push the changes to your course, and apply them on PrairieLearn by clicking "Sync" (on top of your course's PrairieLearn page) and then "Pull from remote Git repository".

After following the steps above, the element is ready to be used in your questions. Please refer to the example question in this course and the README file on Github for more details how to use and customize the element.
