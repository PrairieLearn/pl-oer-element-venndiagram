The questions below were designed to showcase the different features of pl-drawing-venn. The goal of this element is to simplify the rendering of Venn diagrams, and to support interactive questions where students create and shade their own diagrams. This element has many customization options that are documented in the README file in its repository.

Part 1

If the gradable attribute is set to false, Venn diagrams are rendered statically and can be used as instructor-provided material for a question. In this example, 3 circles are drawn and their overlapping area is shaded.
Part 2

By default, Venn diagrams are fully editable, allowing students to add, delete, move, label circles and shade their overlapping areas. The result can be auto-graded based on whether the correct overlapping area between labeled circles is shaded, ignoring the exact placement of each circle. In this example, all regions excluding the area where the circles overlap should be shaded in a correct answer.
Part 3

Because student-defined labels need to match the expected ones exactly, it can be preferrable to pre-define circles and disable re-labeling or adding/removing circles by setting the disable-labeling and disable-insertion attributes to true. Here, the 3 circles need to be moved to overlap, and the overlapping region must be shaded.
Part 4

In this variant of the previous question, moving circles is disabled by setting disable-movement to true. This also disables adding/removing circles, and reduces the degree of freedom for students to only shading the existing circles. Here, the overlapping region between A and B, as well as the entire circle C must be shaded.
